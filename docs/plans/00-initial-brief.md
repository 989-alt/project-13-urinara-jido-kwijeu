# Day 13 · 우리나라 지도 퀴즈 — 시도 위치 맞추기

## 토픽 (100-vibecoding-topics.md #013)
- **카테고리**: 사회과 (4~6학년)
- **콘텐츠 주제·목표**: 무작위로 제시되는 시·도(17개)를 SVG 지도 위에서 클릭으로 맞추기. 한국 행정구역 위치 감각.
- **포함 기능**:
  - 시·도 SVG 지도(GeoJSON 정적 변환)
  - 무작위 출제·즉시 정답 확인
  - 정답률·소요 시간 결과
  - 학습지(시·도 이름 빈 지도) 인쇄
- **배제 기능**: 외부 지도 API 호출(오프라인 동작), 학생 데이터 저장
- **기술 스택**: HTML/CSS/JS, 정적 SVG, GeoJSON 사전 변환
- **저장 방식**: 휘발성 (메모리 한정)
- **AI 옵션**: ✕ (Gemini 미사용)
- **대상**: 4~6학년 · **환경**: TV(전체 시연) + 1:1(개별 학습)
- **DESIGN.md**: Airbnb (지리감) / 보조: Linear

## 구현 제약
- **스택 선택**: 단일 `index.html` + vanilla CSS + vanilla JS (단순 토픽, CDN 0 의존).
- **이유**:
  - SVG 지도 클릭이라는 단일 인터랙션이라 React 도입 오버킬.
  - CDN 의존 0 → 오프라인 동작 보장 (배제 기능 요구).
  - 빌드 단계 없음 → GitHub Pages 즉시 배포.
- **데이터**: 17개 시·도 SVG path는 코드 인라인. GeoJSON에서 사전 변환된 단순화된 path.
- **Gemini 미사용**.

## 핵심 사용자 시나리오
1. 교사가 페이지 진입 → "퀴즈 시작" 버튼.
2. 무작위로 "강원특별자치도를 찾으세요" 같은 질문이 상단에.
3. 학생이 지도 위 영역 클릭 → 정답이면 초록 + 다음 문제, 오답이면 빨강 + 정답 위치 표시 + 다음.
4. 17문제 끝 → 정답률·소요 시간 표시 + 재도전.
5. 별도 "학습지 인쇄" 모드 → 시·도 이름 빈 빈 지도가 A4에 깔끔하게 인쇄.

## 비-기능 요구
- 키보드 접근성 (지도 영역 tabbable, Enter/Space 활성화).
- 색 대비 4.5:1.
- 모바일 / 태블릿 터치 대응 (TV 시연용 대화면 우선).
- 인쇄 CSS (학습지 모드).
