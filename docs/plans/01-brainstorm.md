# Brainstorm — 우리나라 지도 퀴즈

원격 무인 세션이므로 AI 자체 판단으로 결정 (사용자 질의 생략).

## 꼭 필요한 기능 (MUST)

1. **17개 시·도 SVG 지도 + 클릭 정답 판정**
   - 모든 광역시·도(서울, 부산, 대구, 인천, 광주, 대전, 울산, 세종, 경기, 강원특별자치도, 충북, 충남, 전북특별자치도, 전남, 경북, 경남, 제주)를 path로 그려 영역 hover/click 가능.
   - 각 path에 `data-name` 속성으로 정답명 부여, 클릭 시 즉시 채점.

2. **무작위 출제 + 진행 추적**
   - 17문제를 무작위 순서로 1회 출제 (중복 없음).
   - 현재 N번째 / 정답 개수 / 오답 개수 / 진행 표시.

3. **결과 화면 + 재도전**
   - 정답률 (%) + 총 소요 시간 (초 단위) 표시.
   - 오답이었던 시·도 목록 표시 → "오답만 재시험" 버튼.
   - "처음부터 다시" 버튼.

4. **학습지 인쇄 모드**
   - 별도 모드 토글: 시·도 이름이 모두 비어있는(또는 번호만 적힌) A4 인쇄용 지도.
   - `@media print` CSS로 헤더·UI 숨김.

## 있으면 좋은 기능 (SHOULD)

1. **즉시 피드백 애니메이션**
   - 정답: 해당 영역이 0.4초간 초록 깜빡임.
   - 오답: 클릭한 영역 빨강 + 정답 위치 노란 외곽선 1초간.

2. **키보드 접근성 / 힌트 모드**
   - 지도 영역 Tab 순회 가능, Enter/Space로 선택.
   - "힌트 보기" 토글: 시·도 이름 라벨을 지도에 항상 표시 (자유 학습용).

## 절대 금지 기능 (MUST NOT)

1. **외부 지도 API 호출 (Google Maps, Mapbox, Naver Maps 등)**
   - 토픽 명세상 오프라인 동작 요구.
   - 학교 네트워크에서 차단되는 경우 다수 + 키 관리 부담.

2. **학생 데이터 저장 / 서버 전송**
   - 점수·이름·진행 기록을 localStorage 또는 서버에 저장 금지.
   - 토픽 명세상 "휘발성" 저장만 허용. 한 세션 종료 시 데이터 모두 사라짐.

3. **AI 호출 / Gemini 사용**
   - 단순 위치 맞추기 게임에 LLM 불필요. 비용·복잡도 모두 낭비.

4. **시·도 이외의 행정구역(시·군·구 단위) 출제**
   - 17 광역단위로 scope 고정. 시·군·구는 별도 토픽(향후).
   - scope creep 방지.

## 결정 사항 요약
- 단일 HTML / vanilla / CDN 0.
- localStorage 미사용.
- AI 미사용.
- 인쇄 모드 포함.
